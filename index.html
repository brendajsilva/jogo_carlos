Como Jogar

Bem-vindo ao Corrida Lendária! Aqui está o que você precisa saber para começar:

  • Escolha seu carro: Clique em um dos carros disponíveis na tela inicial para selecionar.
  • Controles do Jogador 1: Use as teclas:
    • W: Mover para cima
    • S: Mover para baixo
    • A: Mover para a esquerda
    • D: Mover para a direita
  • Controles do Jogador 2: Use as setas do teclado:
    • ↑: Mover para cima
    • ↓: Mover para baixo
    • ←: Mover para a esquerda
    • →: Mover para a direita
  • Objetivo: Ultrapasse os carros inimigos para ganhar pontos. Evite colisões para não perder vidas.
  • Fases: A cada 20 pontos, a dificuldade aumenta com carros mais rápidos.
  • Game Over: O jogo termina quando ambos os jogadores perdem todas as vidas.
